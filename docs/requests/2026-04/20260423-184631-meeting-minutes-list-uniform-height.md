## 요청 내용

좌측 회의록 리스트에서
1. 모든 아이템이 **동일한 세로 길이**를 가지도록 통일
2. 참여자 아바타 스택 대신 `참여자: 정보영, 강우원, 김홍주` 형태의 **텍스트**로 표시

## 배경

- 현재 아이템은 참석자 유무에 따라 2줄 / 1줄로 달라져서 높이가 들쭉날쭉함
  ([meeting_minutes_screen.dart:1069-1075](lib/screens/meeting_minutes_screen.dart#L1069-L1075))
- 참석자가 있을 때만 아바타 스택 + 이름이 나오는 `_buildAttendeeSummary` 위젯 사용 중
  ([meeting_minutes_screen.dart:1084-1152](lib/screens/meeting_minutes_screen.dart#L1084-L1152))

## 변경 후 ASCII

```
┌────────────────────────────────────────────┐
│ 📄 111                         2026.04.23  │
│     참여자: -                               │
├────────────────────────────────────────────┤
│ 📄 ooooo                       2026.04.23  │
│     참여자: 정보영, 강우원, 김홍주           │
├────────────────────────────────────────────┤
│ 📄 주간회의 정리                2026.04.22  │
│     참여자: 정보영                          │
└────────────────────────────────────────────┘
```

- 모든 행이 **제목행 + 참여자행** 2줄 구조로 고정 → 세로 길이 동일
- 참여자 행: `참여자: 이름1, 이름2, 이름3` (길면 `…` 말줄임)
- 참여자 없을 때: `참여자: -`
- 기존 아바타 스택(`_buildAttendeeSummary`) 제거
