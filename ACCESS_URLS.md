# 접속 주소 가이드

## 🌐 접속 주소 요약

서버 IP: **192.168.1.102**

### 웹 브라우저 접속

```
http://192.168.1.102
```

또는 포트를 명시하면:

```
http://192.168.1.102:80
```

포트 80은 기본 HTTP 포트이므로 생략 가능합니다.

## 📍 각 경로별 접속 주소

### 1. 웹 앱 (Flutter 웹 앱)

```

```

- 메인 웹 애플리케이션
- 로그인 화면부터 모든 기능 사용 가능
- 브라우저에서 바로 사용

```

### 3. API 문서 (Swagger UI)

```

```

또는

```

```

### 4. 헬스 체크

```

## 🔧 플랫폼별 접속 방법

### Windows/macOS 앱 (Flutter 데스크톱)

앱 내부에서 자동으로 다음 주소로 연결:

```
http://192.168.1.102/api/...
```

설정 위치: `lib/utils/api_client.dart`

```dart
static const String baseUrl = 'http://192.168.1.102';
```

### 웹 브라우저

브라우저 주소창에 입력:

```
http://192.168.1.102
```

## 🔐 도메인 사용 (선택사항)

도메인을 사용하려면:

1. 도메인 구매 (예: `dora-project.com`)
2. DNS 설정에서 A 레코드 추가:
   ```
   dora-project.com → 192.168.1.102
   ```
3. Nginx 설정에서 `server_name` 변경:
   ```nginx
   server_name dora-project.com;
   ```

그러면 다음 주소로 접속 가능:

```
http://dora-project.com
```

## ⚠️ 주의사항

### 방화벽 설정

서버에서 포트 80이 열려있는지 확인:

```bash
# Linux/macOS
sudo ufw allow 80

# 또는
sudo iptables -A INPUT -p tcp --dport 80 -j ACCEPT
```

### 같은 네트워크 필요

- 서버와 클라이언트가 같은 네트워크(와이파이/랜)에 있어야 함
- 다른 네트워크에서 접속하려면 포트 포워딩 또는 VPN 필요

### 로컬 접속

서버 자체에서 접속:

```
http://localhost
또는
http://127.0.0.1
```
